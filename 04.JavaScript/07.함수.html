함수

함수를 선언하는 방법

  1. 선언적 함수(Function Declaration) : function키워드, 함수명
  2. 익명 함수(Function Expresssion) : 변수에 익명함수를 대입하는 방식
  3. 화살표 함수(Arrow Function) : 익명함수를 간단히 작성하는 방법(ES6버전부터 추가)
  4. 즉시 실행 함수 iife(Immediately Invoked Function Expresssion)
    • 선언과 호출
    • 변수를 캡슐화처리 유용

* 선언적 함수

        Function 함수명([매개변수,...]) {
            해당 함수 호출시 실행할 코드;
            [return 결과값;]
        }

* 익명 함수

        변수|속성 = function([매개변수,..]) {
            해당 함수 호출시 실행할 코드;
            [return 결과값;]
        }

        - 주로 이벤트 핸들러 작성시 사용

* 화살표 함수(Arrow Function)

        - 본문에 리턴만 있을 때
          변수 = ([매개변수,..]) => 리턴값;

        - 본문이 여러줄 일 때
          변수 = ([매개변수,..]) => {
            해당 함수 호출시 실행할 코드;
            [return 결과값;]
          }

* 즉시 실행 함수

        - 정의하면서 즉시 실행되는 함수
        - 단 한번만 호출됨
          ([변수] = function([매개변수,..]) {
            해당 함수 호출시 실행할 코드;
          })(인자,...);
